Photo Feed plugin is an easy way to display your Instagram Feed within website.

== Description ==

**Instagram media showcase plugin**

Display your Instagram accounts feed within the website as different layouts. Easy to configure and will be ready within a few seconds. Just install and check how it works. it's compatible with all the modern web browsers.

Helpful plugin to add multiple Instagram accounts and to display multiple feeds on the same page.

**How to use:** install and activate like any other plugin, Then go to *Dashboard: Settings > Photo Feed* and connect your Instagram account there. After successful connection, configure new gallery and copy/paste the visible shortcode within pages.

**Demo:** check [Feed demo](https://instaphotofeed.com/) or [Screenshots](#screenshots) section for preview.

== Installation ==

= Minimum Requirements =

* WordPress 5.6+
* PHP version 7.1+

= How to Install =

You can install in the same way you had installed any other plugin or check [instructions here](https://wordpress.org/support/article/managing-plugins/).


== Frequently Asked Questions ==

= Why pictures are not showing on the page? =

Make sure your Instagram account is connected to the website. Or try to re-connect account again (if already connected).

= Why picture links are broken? =

Try to clear cache (if any cache plugin or server-side cache used).
